# Java 学习路线

本仓库用于记录 Java 学习路线，涵盖从大一到大四的完整学习规划。

## 目录结构

- [大一](./year1/README.md) - 全方位学习阶段，你有四年的时间充分准备，这里面的学习路线是最全的，也是最详细的。
- [大二](./year2/README.md) - 由于你是大学二年级才幡然醒悟，所以你的学习路线可以稍微简化一点，因为你的时间已经不多了，所以你需要更加高效的学习，不然你将错过最佳的就业时机。
- [大三](./year3/README.md) - 你更是“重量级选手”了，前面两年过得有多轻松，只有你自己最清楚。那现在怎么办？别想太多，开始学吧。
- [大四](./year4/README.md) - 拉完了，直接拉满负分，成不成只能老天说了算。

## 说明

如果你是在大一下学期开学才开始使用这份路线，那就稍微多给自己一点强度，在这个学期内把整个大一的内容全部走完。

如果你是在大一下学期期末才开始接触这份路线，那就不用再从头慢慢补，大二的学习路线可以直接开始，后面依次类推。

至于已经大四，或者年级在大四以上的同学，如果目标是尽快找到工作，那就直接从大四路线学起，把时间用在最核心、最实战的内容上即可。

任何时候开始学习都不算晚，哪怕你已经大四或者毕业了。记住一句话：你只是大学毕业了，不是死了，人生才刚刚开始。
